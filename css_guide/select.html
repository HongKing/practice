兄弟选择器

h3 + p { color: #f00 } 只有挨着的 p 会应用样式

h3 ~ p { color: #f00 } 所有同级的 p 都会应用样式

简单属性选择器

a[title]{color: #0f0 } 有 title 属性的链接
*[title]{color:#00f;} 包含 title 的所有元素

具体属性选择器

正则匹配选择器

input[class^="txt"]{border:1px solid #f00;} class 以 txt 开头的所有元素

input[class$="er"]{border:1px solid #f00;} class 以 er 结尾的所有元素

input[class*="txt"]{border:1px solid #f00;} class 包含 txt 的所有元素

伪元素/状态选择器

.tip:before{} before 伪元素

.tip:after{} after 伪元素

input:checked{} checked 状态的元素

p:first-letter{font-size: 20px;} 第一个字符

p:first-line{color:#00f;} 第一行

伪元素选择器

链接的状态顺序

按 LVHA 顺序才能正确显示所有状态

a:link{color: #00f;} 未访问过

a:visited{color:#666;} 已经访问过

a:hover{color:#f00;} 鼠标停留在元素时

a:active{color:#ff0;} 鼠标按下激活时

a:visited:hover{color:#0ff;} 组合显示
