字体和字符

font

font: italic 14px Helvetica,Arial,"Microsoft Yahei",sans-serif;

font属性前三个值可以采用任意顺序，后两个值必须font-size在前，font-family在后，同时必须存在

text-indent

文本缩进

text-align

水平对齐

line-height

行高，line-height减font-size，然后再除以2，分别加到顶部和底部

vertial-align

垂直对齐文本，不影响块级元素中内容对齐，但可影响单元表格中元素的垂直对齐，正值使元素上升，负值使元素下降

word-spacing

单词间距

word spacing

letter-spacing

字与字间距

letter spacing

text-transform

文本转换
uppercase 将文本转换为大写
lowercase 将文本转换为小写
capitalize 将单词首字母大写

text-decoration

文本装饰，不能继承

underline

overline

line-through

blink

white-space

控制空白字符的显示

nowrap
nowrap文本不换行强制在一行显示
pre
pre会 保留 额外的空格和 回车，且文本不换行
pre-wrap
pre-wrap会 保留 额外的空格和回车，但文本会换行
pre-line
pre-line，会合并文本中的 空白符，但保留回车， 且文本会换行

word-break

控制单词如何被拆分换行

normal，浏览器默认

This is a incomprehensibilities long word.

keep-all，英文单词强制一行显示，只有空格可以触发自动换行

This is a incomprehensibilities long word.

break-all，英文单词强制内部换行，所有单词碰到边界一律拆分换行

This is a incomprehensibilities long word.

word-wrap/overflow-wrap

控制长度超过一行的单词是否被拆分换行，又叫做overflow-wrap

word-wrap 属性原本属于微软的一个私有属性，在 CSS3 现在的文本规范草案中已经被重名为 overflow-wrap 。 word-wrap 现在被当作 overflow-wrap 的 “别名”。

normal

break-word，只有当一个单词一整行都显示不下时，才会拆分换行该单词
